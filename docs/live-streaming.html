Cloudinary Video Player

Live Streaming

Provide your cloud name and active live streaming url to see output directly inside Video Player

Full documentation

Example Code:

      
        <video id="player" class="cld-video-player"></video>
      
      

        const player = cloudinary.videoPlayer('player', {
          cloudName: 'YOUR_CLOUD_NAME',
          publicId: 'LIVE_STREAMING_URL',
          type: 'live',
          controls: true,
          sourceTypes: ['hls'],
        });
