Breakpoints - Responsive Video Resolution

Breakpoints automatically select the optimal video resolution based on container width and device pixel ratio (DPR).

With Breakpoints (DPR 2.0)

Automatically serves optimal resolution based on player size and device DPR.

View Code 
const player = cloudinary.videoPlayer('player', {
  cloudName: 'demo',
  breakpoints: true,
  dpr: 2.0
});

player.source('sea_turtle');

Without Breakpoints (Comparison)

Default behavior - serves original video dimensions.

View Code 
const player = cloudinary.videoPlayer('player', {
  cloudName: 'demo'
});

player.source('sea_turtle');

Configuration Options

  • breakpoints (boolean): Enable/disable breakpoints
  • dpr (number): Device pixel ratio - 1.0, 1.5, or 2.0 (default: 2.0)
