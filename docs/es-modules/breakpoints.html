Breakpoints - Responsive Video Resolution

Breakpoints automatically select the optimal video resolution based on player width and device pixel ratio (DPR). Container width is rounded to the nearest rendition [640, 1280, 1920, 3840] and Cloudinary handles DPR scaling.

Configuration

  • breakpoints (boolean): Enable/disable breakpoints
  • dpr (number): Device pixel ratio - 1.0, 1.5, or 2.0 (default: 2.0)
  • Renditions: [640, 1280, 1920, 3840]
